lews-homepage Rejects
What is this? It is a personalised homepage for me, lewis. Biorxiv, Chemrxiv, and RCSB Entries from the last seven days have been downloaded, then filtered by a cheap model with a prompt intended to catch articles interesting to me. Articles can be ticked if they've been seen, to avoid re-reading them, which is stored per-browser using localStorage. Articles are updated weekly on LJM's computer with a cronjob and pushed back to github. The repo is at https://github.com/ljmartin/lews-homepage
